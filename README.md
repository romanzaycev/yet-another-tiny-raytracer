# yet-another-tiny-raytracer

_Based on Dmitry V. Sokolov's "tinyraytracer" project_ 


###### Build and run:

```bash
git clone https://github.com/romanzaycev/yet-another-tiny-raytracer.git
cd yet-another-tiny-raytracer
make run
```
